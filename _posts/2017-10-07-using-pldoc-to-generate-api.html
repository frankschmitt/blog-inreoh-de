Sample call
pldoc.sh -url jdbc:oracle:thin:@//hostname:port/servicename -user -password -sql

e.g.

pldoc.sh -url jdbc:oracle:thin:@//localhost:1521/xe -user scott -password tiger -sql sys.dbms_output
